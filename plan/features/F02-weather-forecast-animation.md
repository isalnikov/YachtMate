# F02 — Прогноз погоды и анимированная карта (класс Windfinder / Windy)

**Источник идей:** Windfinder, Windy (windy.com); расширенный блок IDEAS: Clime, Weather radar, Windy.app (сообщества по спотам — см. Фазу 6), WindCompass; рекомендации PocketGrib/ZyGrib как офлайн-GRIB ([F10-offline-grib.md](F10-offline-grib.md)).

## Возможности референса

| Приложение | Ключевые фишки |
|------------|----------------|
| Windfinder | GFS ~27 км (free), платно ~12 км; до 3 пользовательских точек; 7 дней; почасовой прогноз (paid); порывы ветра, волны, температура, осадки, течения; таблица + анимированная карта |
| Windy.com | Анимированная карта, цвет по силе ветра; точка пальцем → почасовой прогноз; до 10 дней; офлайн-кэш; несколько слоёв и **сравнение моделей** прогноза; изобары; рекомендованные споты; часто — веб-камеры на точках |
| Windy.app (отдельный продукт в IDEAS) | Офлайн-карты; аккаунт; **комьюнити по спотам**, чаты, фото — социальный слой у погоды |
| Clime | Прогноз на неделю; уведомления при резких изменениях температуры/осадков; молнии/ураганы; карты и инфографика |
| Weather radar | Усиленная детализация ветра в базовой версии; циклоны/антициклоны, штормы |
| WindCompass | Порывы как фокус; настройка магнитный/истинный курс; направление ветра «откуда/куда»; UV, рассвет/закат |

## План реализации фичи

1. **Источники данных:** Open-Meteo / ECMWF через доступные API; абстракция провайдера для смены модели; UI **переключения модели** и подписи источника (идея Windy).
2. **Точки интереса:** сохранённые локации (лимит настраиваемый — аналог «3 точки» Windfinder на бесплатном тарифе продукта).
3. **Горизонт и шаг:** 7–10 дней конфигурируемо; почасовой ряд для выбранной точки; акцент на **порывах** при отображении ветла (WindCompass-like опции).
4. **Переменные:** скорость и порывы ветра, направление, высота волн, осадки, давление, качество воздуха при наличии API; поверхностные течения; опционально изобары на карте.
5. **Представление:** экран таблицы/графиков + слой карты с полем ветра и временным слайдером (MVP упрощённый Windy-like).
6. **Уведомления:** пороги по ветру/осадкам/молниям (если регион и API допускают); не дублировать национальные сервисы без явной интеграции — см. разрешения и юридику региона.
7. **Офлайн:** кэш последнего успешного прогноза по bbox/точке с меткой времени (см. фазу 4); для полноценной работы без интернета в море — дорожная карта на импорт GRIB ([F10-offline-grib.md](F10-offline-grib.md)).
8. **Сообщества по спотам:** не смешивать с ядром погоды — вынос в [phase-06-marinas-community.md](../phase-06-marinas-community.md) и [F05-marinas-anchorages-booking.md](F05-marinas-anchorages-booking.md).
9. **Локализация:** RU/EN с первого релиза погодного модуля.

## Критерии приёмки

- Для сохранённой точки доступен почасовой прогноз без сети после кэширования (в пределах TTL).
- Анимация слоя не блокирует взаимодействие с картой (отключаемый слой).
- Единицы (узлы/м/°C) согласованы с настройками приложения.
