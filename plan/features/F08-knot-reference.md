# F08 — Справочник морских узлов (класс «Справочник узлов»)

**Источник идей:** «Справочник узлов»; Knot Guide (игровой формат, рыбацкие и «галстучные» узлы, глоссарий частей снасти) в [`../../IDEAS.md`](../../IDEAS.md).

## Возможности референса

- Более 100 узлов, структурированных по типу применения (яхтенные, рыбацкие и др.).
- Иллюстрации пошагового вязания (в IDEAS также упоминаются схемы vs видео — для MVP статика).
- Полностью офлайн; опционально геймификация по мотивам Knot Guide.

Связанный образовательный контент: [F12-maritime-training-radio-colreg.md](F12-maritime-training-radio-colreg.md), [F13-medical-glossary-reference.md](F13-medical-glossary-reference.md).

## Связь с фазами

- Независимый модуль контента; можно ввести после [phase-00-bootstrap.md](../phase-00-bootstrap.md).

## План реализации фичи

1. **Контент:** JSON/база с полями: название (i18n), категория (связь, швартовка, аварийные…), шаги, SVG/растр иллюстраций; лицензия на изображения.
2. **UI:** список по категориям, поиск, избранное.
3. **Офлайн:** весь справочник в бандле или скачиваемый пакет однократно.
4. **Тесты:** отсутствие битых ссылок на ассеты; локализация ключевых узлов RU/EN.

## Критерии приёмки

- Пользователь находит узел по названию и открывает пошаговую инструкцию без сети.
