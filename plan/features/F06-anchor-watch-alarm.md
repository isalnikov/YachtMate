# F06 — Якорная вахта и тревога (класс Anchor Alarm / Anchor Mate)

**Источник идей:** Anchor Alarm / Anchor Mate / Anchor watch lite; Zenkou — в [`../../IDEAS.md`](../../IDEAS.md).

## Возможности референса

| Режим | Описание |
|-------|----------|
| Базовый | Опасная зона относительно точки якоря; тревога при выходе за пределы |
| Anchor watch lite (IDEAS) | Точка якоря по **азимуту и дистанции** от лодки; круговой сектор качания; тревога при **потере GPS** |
| Zenkou | Зона не только радиусом, но и **контуром пальцем**; опционально **обмен радиусом с соседними лодками**; интеграция с картами и AIS |
| Общее | Громкий сигнал; фоновая работа; рекомендации IDEAS: питание, экран всегда включён |

## Связь с фазами

- [phase-01-navigation-core.md](../phase-01-navigation-core.md) — точный GPS.

## План реализации фичи

1. **Постановка:** пользователь фиксирует «якорь опущен» → базовая точка + радиус ИЛИ многоугольник (post-MVP: произвольный контур как Zenkou).
2. **Альтернативная постановка:** якорная точка по bearing+range от текущей позиции судна (якорь не над точкой GPS).
3. **Мониторинг:** периодическое чтение позиции; расстояние до границы зоны; гистерезис против дрожания GPS.
4. **Потеря фикса GPS:** отдельная тревога «нет позиции» с подавлением ложных срабатываний при кратковременном провале.
5. **Тревога:** полноэкранный / максимальный звук + вибрация; кнопка подтверждения «всё в порядке».
6. **Фон:** foreground service на Android; на iOS — совместимо с политиками фонового геопозиционирования и уведомлением о вахте.
7. **Расход батареи:** режим интервала опроса (энерго-экономия из [phase-08-polish.md](../phase-08-polish.md)).
8. **Соседи (опционально):** обмен только радиусом/контуром при явном opt-in и без передачи персональных данных — отдельная политика приватности.
9. **Интеграция AIS:** если доступен локальный AIS ([phase-03-ais-nmea.md](../phase-03-ais-nmea.md)), показ соседей для контекста — не путать с Zenkou «шарингом».

## Критерии приёмки

- Симуляция выхода за радиус инициирует тревогу на тестовом стенде.
- Документированы ограничения точности GPS в узких бухтах.
