# F13 — Первая помощь, морской жаргон и справочные шкалы

**Источник идей:** «Первая помощь» (Красный Крест); 7ft (жаргон, флаги стран, отрывки МППСС-72, IMO фразы); cMate-lite (шкалы Бофорта, видимости, состояния моря; кодексы; англо-русские фразы для прогнозов и координат; символы IMO) в [`../../IDEAS.md`](../../IDEAS.md).

## Связь с фазами

- Медицина и экстренные номера пересекаются с [phase-07-safety-logbook.md](../phase-07-safety-logbook.md) — здесь **обучение**, не телемедицина.
- COLREG-картинки частично в [F12-maritime-training-radio-colreg.md](F12-maritime-training-radio-colreg.md).

## План реализации фичи

1. **Первая помощь:** адаптированные карточки (аптечка, кровотечение, удушье, тепловой удар и т.д.); вызов экстренных служб по региону через платформенный API; дисклеймер «не замена курсу медика».
2. **Глоссарий:** термины и жаргон; поиск; опционально юмористические примеры без вреда для безопасности.
3. **Шкалы:** Бофорта, волнение моря, видимость — таблицы + иллюстрации.
4. **Фразы IMO / прогнозы:** двуязычные шпаргалки для радио и чтения NAVTEX-подобных текстов.
5. **Символы:** пожарные/запретные/предупреждающие знаки на судне — галерея по категориям (как в IDEAS про cMate).

## Критерии приёмки

- Раздел первой помощи полностью доступен офлайн после установки.
- Любой совет по медицине сопровождается напоминанием обращаться к профессионалам при серьёзных симптомах.
