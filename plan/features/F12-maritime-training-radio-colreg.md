# F12 — Тренажёры УКВ, COLREG и световые сигналы

**Источник идей:** VHF talk, VHF trainer, Maritime VHF Radio Operator; COLREG 72: safety at sea; фрагменты cMate-lite в [`../../IDEAS.md`](../../IDEAS.md).

## Возможности референса

| Область | Фишки |
|---------|--------|
| УКВ | Аудиотренажер фраз RU/EN; сценарии марина, береговая охрана, MAYDAY/PAN PAN; симуляция на «воде» |
| Регламент | От SECURITE до OVER; запись своего голоса и сравнение с эталоном |
| COLREG | Огни и знаки; яркие схемы; тренажёр по столкновению правил |
| Сигналы | Звуковые и световые сигналы по ситуациям; поиск судна по изображению ночью (cMate) |

## Связь с фазами

- Контентный/образовательный модуль; после [phase-00-bootstrap.md](../phase-00-bootstrap.md).
- Не путать с реальной передачей MAYDAY из [phase-07-safety-logbook.md](../phase-07-safety-logbook.md) — только обучение.

## План реализации фичи

1. **УКВ теория:** короткие карточки + тест с вариантами ответов (уровни шкипер / «большое судно» как в референсе).
2. **Фонетический алфавит и флаги:** прослушивание эталона, запись пользователя (опционально).
3. **Сценарии:** ветвление диалога; оценка не по ИИ на MVP — чеклист ключевых слов.
4. **COLREG:** банк изображений огней/знаков; режим викторины; ссылки на официальный текст (локализованный обзор, не замена документа).
5. **Офлайн:** весь контент обучения в пакете приложения.

## Критерии приёмки

- Пользователь проходит минимум один сценарий УКВ и одну викторину COLREG без сети.
