# Фаза 9 — Справочник узлов (F08 / M9 обучение)

**Цель:** офлайн-справочник морских узлов после стабилизации MVP+ ([`features/F08-knot-reference.md`](features/F08-knot-reference.md)): список по категориям, поиск по названию, карточка узла с пошаговым текстом без сети.

**Зависимости:** Фаза 8 (локали/настройки); контент — ассет JSON + строки интерфейса в ARB.

**Выход:** [`assets/reference/knots_demo.json`](../mobile/assets/reference/knots_demo.json), репозиторий/экраны в [`mobile/lib/features/reference/`](../mobile/lib/features/reference/), вход из вкладки «Ещё», аудит открытия узла, тесты парсера и потока данных.

---

## Критерии приёмки

- [x] Узлы загружаются из бандла; без сети доступны список и деталь.
- [x] Язык шагов следует локали приложения (`en` / `ru` и др.) с запасным **en**.
- [x] Действие открытия узла попадает в `user_action_audit`.
- [x] `flutter analyze` и `flutter test` зелёные.
