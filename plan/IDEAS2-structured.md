# IDEAS2 — структурированные вводные (без дублей)

Исходник: [`../IDEAS2.md`](../IDEAS2.md). В исходном файле одни и те же темы повторяются под разными заголовками (9 блоков в начале, затем таблицы SWOT, затем разделы 1–8, затем модули 1–10, Freemium-пирамида). Ниже — **единая каноническая модель** для Captain Wrongel.

## 1. Переосмысление продукта

**Позиционирование:** один карманный «штурман» с **offline-first**, объединяющий навигацию (как Navionics/iNavX), метео (Windy/Windfinder), социальную лоцию (Navily), безопасность якоря и AIS, обучение (УКВ/COLREG/узлы) и учёт похода — без требования «скопировать 1000 фич»: приоритет **надёжность и скорость работы на палубе** над количеством экранов.

**Дубли, сведённые в один модуль:**

| Повторяется в IDEAS2 под разными именами | Единый модуль |
|------------------------------------------|----------------|
| Приливы в блоке «Погода» и отдельные приложения Tide* | **Гидрология** внутри метео-ветки + отдельный экран «Приливы» при необходимости UX |
| Марины / Marinatips / MySea / No Foreign Land | **Стоянки и берег** (POI + социальные слои) |
| Anchor Alarm / Zenkou / Mate | **Якорная вахта** (один экран, режимы: круг / контур / соседи) |
| Журнал SailGrib / Sail Expert / расходы | **Яхта: журнал и учёт** |
| VHF Talk / Trainer / Maritime Operator | **Радио и SMCP** |
| COLREG / cMate / 7ft | частично **Обучение**, частично **Справочники** |

**Новые акценты IDEAS2 относительно IDEAS.md:**

- Отдельный крупный блок **«Планирование маршрута»** (конструктор, сценарии «что если», библиотека GPX).
- Явный **Freemium / подписка** (базовый уровень vs продвинутый) — отразить в paywall и feature flags.
- **VHF + ИИ-проверка произношения** как УТП для СНГ — вынести в дорожную карту после базового тренажёра.
- **Savvy Navvy / Aqua Map / PredictWind / Blitzortung** — как референсы возможностей (планировщик выхода, NMEA/Signal-K, роутинг по погоде, молнии), не как отдельные пункты меню 1:1.

## 2. Каноническая модульная архитектура (14 модулей → UI)

Каждый модуль имеет экран или вложенный раздел в приложении.

| № | Модуль | Суть | Примечание |
|---|--------|------|------------|
| M1 | **Карта и навигация** | Векторные/растровые слои, GPS, трек, инструменты штурмана, Auto Guidance (advisory) | Ядро |
| M2 | **Планирование маршрута** | Waypoints, GPX, анализ рисков, сценарии «что если», библиотека маршрутов | Новый топ-уровень по IDEAS2 |
| M3 | **Погода и среда** | Мультимодель, ветер/волны, экстремальные явления, опционально слой молний | |
| M4 | **Приливы и течения** | HW/LW, атлас течений, астрономия для планирования | Может быть вкладкой в M3 или отдельным входом |
| M5 | **Стоянки и берег** | Марины, якорные, фильтры, отзывы, бронирование, Marinatips-style POI | Community-часть пересекается с M10 |
| M6 | **Якорная вахта** | Геозона, тревога, SMS при критическом срыве (опционально), Zenkou-style | Safety |
| M7 | **AIS и трафик** | Локальный NMEA + опционально облако, CPA/TCPA, поиск, AR post-MVP | |
| M8 | **Связь и УКВ** | Тренажёр, SMCP, фразы; без подмены реальной VHF без сертификации | |
| M9 | **Обучение и справочники** | Узлы, COLREG, первая помощь, глоссарий, cMate-like шкалы | |
| M10 | **Сообщество** | Отзывы, форумы, маршруты сообщества, события | Модерация |
| M11 | **Яхта: журнал и учёт** | Трек, топливо, расходы, ТО, инвентарь, документы | |
| M12 | **Безопасность и ЧС** | SOS, мониторинг рейса, чеклисты, энергосбережение аварийное | |
| M13 | **Инструменты офлайн** | Компас, астро, GRIB, PDF, офлайн LLM | |
| M14 | **Настройки и экосистема** | Судно, темы, синхронизация, интеграции NMEA | |

**Кросс-срезы:** «Умный капитан» (агрегация данных модулей), единый офлайн-режим, адаптивный UI — не отдельные пункты нижнего меню, а **режимы и настройки**.

## 3. Соответствие MVP из IDEAS2

Первые три для безупречной реализации:

1. **Навигация (карты)** — офлайн, глубины, маршрут от руки.  
2. **Погода** — 3 дня, быстрая анимация слоя ветра.  
3. **Якорь** — будильник + «я на якоре» быстрый режим.

Остальное — модули подписки / следующие релизы.
