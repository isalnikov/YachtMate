# Фаза 1 — Ядро навигации и карты (MVP)

**Цель:** работающая карта, позиционирование в реальном времени, офлайн-регион, минимальный маршрут (точки/линия), юридический дисклеймер.

**Зависимости:** [Фаза 0](phase-00-bootstrap.md) завершена.

**Ссылки:** [`../TODO.md`](../TODO.md), **[`../Подробный план реализации.md`](../Подробный%20план%20реализации.md)** (модуль M1, аудит Приложение A).

---

## Связь с финальным планом

- **TDD:** сначала тесты для доменной логики GPS (сглаживание COG/SOG, loss of fix), геометрии маршрута, репозитория маршрутов; затем реализация.
- **Аудит:** `disclaimer_accept` (при первом согласии), `route_save` / `route_delete`, `chart_region_mount` / `chart_region_remove` (см. финальный план §10.2, Приложение A).
- **Логирование:** агрегированные события потери GPS (не каждая точка); ошибки загрузки тайлов — `level` + `attrs` без PII.

---

## Шаги и артефакты

| Шаг | Действие | Выход |
|-----|----------|--------|
| 1.1 | Интеграция MapLibre GL (или утверждённого движка); базовый стиль карты | Экран карты |
| 1.2 | Поток GPS: координаты, скорость SOG, курс COG, точность HDOP/accuracy; деградация при потере фикса | Use case + юнит-тесты логики |
| 1.3 | Офлайн: MBTiles или векторные тайлы; выбор области; прогресс загрузки; лимиты хранилища | Загрузчик + менеджер кэша |
| 1.4 | Черновой маршрут: waypoints, polyline; без расчёта безопасных глубин (это Фаза 5) | Слой маршрута |
| 1.5 | Дисклеймер при первом запуске и в настройках (не ECDIS/радар) | UI + тексты локализации |

## Детализация

### 1.2 GPS

- Обработать: отсутствие разрешений, временный loss of fix, фоновый режим (политики iOS/Android).
- Настройка частоты обновления для энергосбережения (расширяется в Фазе 8).

### 1.3 Офлайн

- Явное отображение занятого места; удаление региона.
- Источник тайлов для MVP — согласованный в Фазе 0 (open style / собственный кэш).

### 1.4 Маршрут

- Сохранение в локальной БД; экспорт позже при необходимости.

---

## Критерии приёмки

- Pan/zoom карты плавные; позиция судна обновляется при доступном GPS.
- Загруженный регион отображается в авиарежиме.
- Нет краша при выключении GPS или отзыве разрешений (корректное состояние UI).

---

## Риски

| Риск | Митигация |
|------|-----------|
| Различие COG/SOG на малой скорости | Сглаживание/порог в доменной логике |
| Размер офлайн-пакета | Сжатие, выбор области пользователем |
