# Фаза 2 — Глубины, буи, слои карты

**Цель:** отображение изобат и навигационных объектов (буи, маяки — по доступным данным) с переключением слоёв и контекстной информацией по long-press.

**Зависимости:** [Фаза 1](phase-01-navigation-core.md) (карта и офлайн-база).

**Ссылки:** **[`../Подробный план реализации.md`](../Подробный%20план%20реализации.md)** (модуль M1 слои).

---

## Связь с финальным планом

- **TDD:** тесты на переключение слоёв, парсинг метаданных слоя, отсутствие слоя в кэше (fallback UI без исключений).
- **Аудит:** события включения/выключения слоя глубин/буёв (`layer_toggle` с id слоя в `context_json` без больших payload).
- **Комментирование:** в коде декодеров геометрии — ссылка на лицензию источника (`data-licensing.md`).

---

## Шаги и артефакты

| Шаг | Действие | Выход |
|-----|----------|--------|
| 2.1 | Выбор **легального** источника данных: контракт с провайдером и/или открытые данные с допустимой лицензией | Документ `data-licensing.md` |
| 2.2 | Пайплайн: сырьё → векторные тайлы или растровый overlay совместимо с MapLibre | Скрипты/CI шаг генерации |
| 2.3 | UI: список слоёв, видимость, порядок, легенда глубин | Панель слоёв |
| 2.4 | Long-press: координаты WGS84, глубина/высота суши если есть в источнике | Bottom sheet или аналог |

---

## Детализация

### 2.1 Лицензирование

- ENC/S-57 обычно требует IHO/национальных лицензий — для MVP возможен упрощённый слой с явным ограничением точности в UI.
- Задокументировать ограничения для пользователя (splash / справка).

### 2.2 Производительность

- Упрощение геометрии на мелких зумах; LOD.

## Риски

| Риск | Митигация |
|------|-----------|
| Неполные данные без ENC | Прозрачное описание; roadmap обновления слоёв |
| Большой размер пакетов | Региональная загрузка как в Фазе 1 |

---

## Критерии приёмки

- Пользователь может включить/выключить слой глубин и видеть легенду.
- Long-press показывает координаты; глубина — если присутствует в данных.
- Офлайн: заранее загруженные слои работают без сети.

## Чек-лист перехода к Фазе 3

- [ ] Юридически чистый источник для текущего рынка/региона MVP.
