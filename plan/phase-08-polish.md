# Фаза 8 — Производительность, доступность, полировка

**Цель:** соответствие блоку «Производительность и UX» из [`../TODO.md`](../TODO.md): режим перчатки, крупный шрифт, высокий контраст; энергосбережение GPS и карты; поддержка планшетов, поворота, split-view; расширение локализаций (DE, FR, ES, IT) после стабилизации RU/EN.

**Зависимости:** все основные функции предыдущих фаз в рабочем состоянии для целевого релиза.

**Ссылки:** [`../TODO.md`](../TODO.md), **[`../Подробный план реализации.md`](../Подробный%20план%20реализации.md)** (§9 качество, локализация, пороги покрытия).

---

## Связь с финальным планом

- **TDD / качество:** golden-тесты ключевых экранов где уместно; регрессия доступности (контраст, размер hit-target).
- **Логирование:** для release — согласованные уровни (`info` сценарии, `warning` деградации, `error` с `error_code`).
- **i18n:** новые локали только через ARB и `flutter gen-l10n`; проверка переполнения на планшетах и длинных строках.

- Чеклист доступности пройден (минимальный набор WCAG-подобных критериев для мобильных).
