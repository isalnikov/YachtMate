# Scope MVP (Фазы 0–1, ориентир)

Цель первого релиза магазина после завершения **Фазы 1** по [`plan/phase-01-navigation-core.md`](../plan/phase-01-navigation-core.md):

- Карта (MapLibre), GPS, офлайн-регион, черновой маршрут, юридический дисклеймер.
- RU/EN UI, offline-first для загруженных данных региона.
- Инфраструктура аудита и логирования (Фаза 0) включена до пользовательских сценариев после дисклеймера.

Функции Фазы 2+ не входят в MVP unless явно помечены как блокер.
