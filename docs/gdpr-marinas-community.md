# GDPR / персональные данные — марины, отзывы, Crew Match (Фаза 6)

Краткая политика для MVP Captain Wrongel (не юридическая консультация).

## Локальные черновики отзывов

- Полный текст комментария хранится только в локальной базе (**`mooring_review_drafts`**) до появления сервера синхронизации.
- В **`user_action_audit`** при сохранении черновика не записывается текст отзыва — только безопасный контекст (`placeId`, `stars`), чтобы соблюсти минимизацию данных при расследованиях.

## Будущая синхронизация

- При включении выгрузки потребуется явное согласие пользователя и описание цели обработки в политике конфиденциальности приложения.
- Crew Match и профили — отдельное согласие и документирование правовых оснований по GDPR до включения функций в продакшене.
