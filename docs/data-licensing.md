# Captain Wrongel — источники картографических данных (MVP)

Документ фиксирует **легальный статус** демонстрационных слоёв и базовой подложки, чтобы выполнить требования [фазы 2](../plan/phase-02-bathymetry-layers.md) (п. 2.1).

## Базовая карта (MapLibre)

- Для прототипа используется публичный стиль `**https://demotiles.maplibre.org/style.json`** (`MapLibreStyles.demo` в приложении).
- Условия использования и атрибуция — по лицензии проекта MapLibre / поставщика тайлов, указанного в самом стиле. Приложение **не заявляет** статус ECDIS или официальной ENC.

## Слой изобат и расчёт advisory-сетки (Фаза 2 / 5)

### Активный слой (`mobile/assets/map/fethiye_navigation_layers.geojson`)

- Контуры получены через **OGC WFS** сервиса **EMODnet Bathymetry** (`http://ows.emodnet-bathymetry.eu/wfs`, слой `emodnet:contours`), выборка по ограничивающему прямоугольнику в районе **бухты Фетхие (Турция)** (пример для разработки).
- Свойство глубины в источнике (`elevation`) при сохранении в репозитории приведено к полю **`depth_m`** (положительное число, метры).
- Условия использования данных EMODnet — на [EMODnet Bathymetry](https://emodnet-bathymetry.eu/) и в метаданных продукта; приложение использует данные только как **иллюстрацию** возможностей слоя и **не** заменяет официальные морские карты или съёмку IHO.

### Учебный GeoJSON (`mobile/assets/map/demo_navigation_layers.geojson`)

- Оставлен в репозитории как **маленький синтетический** пример для тестов и документации; по умолчанию карта и маршрутизация используют файл Фетхие выше.

Реальные ENC/S-57 для производственного использования требуют отдельных лицензий и пайплайна преобразования — см. дорожную карту в `plan/phase-02-bathymetry-layers.md`.

## Пользовательский текст

В интерфейсе сохраняется формулировка: приложение не является ECDIS; при отсутствии загруженного слоя показывается корректная деградация без падений.
