# Captain Wrongel — источники картографических данных (MVP)

Документ фиксирует **легальный статус** демонстрационных слоёв и базовой подложки, чтобы выполнить требования [фазы 2](../plan/phase-02-bathymetry-layers.md) (п. 2.1).

## Базовая карта (MapLibre)

- Для прототипа используется публичный стиль **`https://demotiles.maplibre.org/style.json`** (`MapLibreStyles.demo` в приложении).
- Условия использования и атрибуция — по лицензии проекта MapLibre / поставщика тайлов, указанного в самом стиле. Приложение **не заявляет** статус ECDIS или официальной ENC.

## Демо-слой глубин и навигационных объектов (Фаза 2)

- Данные поставляются как **локальный GeoJSON** в репозитории (`mobile/assets/map/demo_navigation_layers.geojson`).
- Это **синтетические / учебные** линии изобат и точки условных знаков для региона демо-камеры (не гидрографический съёмка IHO).
- Глубины и названия объектов **не предназначены для навигационных решений** и отображаются только как пример возможности слоёв и запроса по long-press.

Реальные ENC/S-57 для производственного использования требуют отдельных лицензий и пайплайна преобразования — см. дорожную карту в `plan/phase-02-bathymetry-layers.md`.

## Пользовательский текст

В интерфейсе сохраняется формулировка: приложение не является ECDIS; при отсутствии загруженного слоя показывается корректная деградация без падений.
